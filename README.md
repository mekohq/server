## 📝 Installation & Setup

### Prerequisites
- Node.js v14+
- npm or yarn
- Discord Account

### Get Started

```bash
# Clone the repository
git clone https://github.com/mekohq/server.git

# Install dependencies
npm install

# Configure environment variables
cp .env.example .env
# Edit .env with your Discord token

# Start the bot
npm start
```

## 🤖 Usage Examples
